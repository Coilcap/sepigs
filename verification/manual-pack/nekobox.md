# NekoBox

Create a SOCKS profile using `SEPIGS_HOST:1080` and configured inbound credentials. Start the profile and open `https://example.com/`. Success requires a loaded page and a matching Dashboard connection. Check LAN reachability, Android VPN permission, battery restrictions, firewall, and credentials on failure. Stop the profile to roll back.
