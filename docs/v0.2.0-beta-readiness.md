## Release Decision

The codebase meets the conditions for a public `v0.2.0-beta.0` prerelease:
automated gates, the full 24-hour soak, and user manual client signoff passed.
External protocol compatibility remains explicitly blocked rather than
claimed. This is not production-stable. The existing tag must remain unchanged,
and no `v0.2.0-beta.1` is required by the current evidence.
